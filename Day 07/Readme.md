# Simple Python Programs

A collection of simple, beginner-friendly Python scripts. Each file is standalone and can be run directly.

## Files

| File | Description |

| `cube.py` | Calculates the cube of a number. |
| `area_calci.py` | Calculates the area of a square, rectangle, circle, or triangle. |
| `vowel_counting.py` | Counts the number of vowels in a string. |
| `palindrome.py` | Checks whether a given word, phrase, or number is a palindrome. |
| `max_min.py` | Finds the maximum and minimum values in a list of numbers. |
| `student_result_system.py` | Calculates a student's total marks, percentage, and grade. |
| `password_strength_checker.py` | Checks how strong a password is and gives improvement suggestions. |

## How to Run

Each script can be run with Python :

```bash
python cube.py
python area_calculator.py
python vowel_counter.py
python palindrome.py
python max_min.py
python student_result_system.py
python password_strength_checker.py
```

## Requirements

- Python
- No external libraries required (only built-in modules like `math` and `re` are used).
